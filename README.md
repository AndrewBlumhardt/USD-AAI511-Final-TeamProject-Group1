# USD AAI-511 Summer 2026 Final Team Project

## Classical Music Composer Classification Using Deep Learning

This repository contains the source code, Jupyter notebooks, supporting
documentation, and final report for the University of San Diego
**AAI-511** Summer 2026 group project.

The objective of this project is to design, implement, and evaluate two
deep learning approaches for identifying the composer of a classical
music piece from symbolic MIDI data. The project compares the
performance of a Long Short-Term Memory (LSTM) network and a
Convolutional Neural Network (CNN) using standard machine learning
evaluation metrics.

## Project Objectives

-   Prepare and preprocess a labeled MIDI dataset.
-   Extract meaningful musical features.
-   Develop and train LSTM and CNN models.
-   Compare model performance using accuracy, precision, recall, F1
    score, and confusion matrices.
-   Document the methodology and results in both a Jupyter Notebook and
    an APA-style report.
